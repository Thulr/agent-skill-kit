# Structure Rubric

The organize-the-ideas method. Used by `structure` (build it) and `diagnose`
(find what's broken). Operates above the sentence — on the order and grouping
of points.

## Lead with the answer (top-down)

State the governing conclusion or recommendation **before** the support.
Invert the order in which you *discovered* the answer (bottom-up: evidence →
conclusion) when you *communicate* it (top-down: conclusion → evidence). A
reader who agrees can stop early; a skeptic stays anchored to what you're
defending.

## Open with SCQA

Frame the intro as **Situation** (context the reader already accepts) →
**Complication** (the change or tension that triggers the piece) →
**Question** (what the complication raises) → **Answer** (your governing
thought). Drafting the SCQA first surfaces the question the reader actually
has before you write the body.

## Group MECE

Within any group of supporting points: **Mutually Exclusive** (no overlap),
**Collectively Exhaustive** (no gaps), usually 3–5 items. The group's heading
must **assert the insight** ("Latency comes from the cold cache, not the
query"), not name a placeholder ("Three findings"). Pick deductive *or*
inductive ordering within a group, not both.

## Does this section turn?

Every section should move a **named value** start→end (unknown→known,
confident→doubtful, working→broken). A section where nothing changes state is
a cut-or-merge candidate. As a diagnostic for a flat draft, walk each
section: inciting disturbance → complication → the turn → resolution. Missing
the disturbance or the turn is why a section "drags."

## Causal necessity, not episodic listing

Each point should follow the prior as a necessary or probable consequence —
the cure for a draft that reads "and then, and then." If points are merely
sequential, either make the causation explicit or admit it's a list (and then
an answer-first structure beats a forced narrative — see
[`narrative-honesty-guard.md`](./narrative-honesty-guard.md)).

## Unity test

Cut anything whose removal leaves the whole intact. If a paragraph can go and
nothing downstream breaks, it wasn't load-bearing.
