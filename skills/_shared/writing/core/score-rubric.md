# Score Rubric

Use this 0–10 scale only when the `diagnose` intent is asked to **score** a
piece (clarity, structure, or stickiness). Always name the dimension you are
scoring; do not collapse several dimensions into one number.

| Band | Meaning |
| --- | --- |
| 9–10 | Ships as-is. Purpose is unmistakable; a cold reader gets it the first time. |
| 7–8 | Solid; a light revise pass would sharpen it. No structural problem. |
| 5–6 | Readable but the point is worked-for — usually a structure or ordering problem, not a line problem. |
| 3–4 | The reader is likely to miss the point or the action. Needs reorganizing, not polishing. |
| 0–2 | Fails its purpose for the intended reader. |

**Caps.** Any single severity-4 finding caps the score at 4 — a piece that
buries the one decision it exists to surface is not a 7 because the sentences
are pretty. Score the reader's experience of the whole, not the average of
the parts.
