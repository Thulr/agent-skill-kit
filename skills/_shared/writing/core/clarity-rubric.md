# Clarity Rubric

The line-and-paragraph method. Used by `revise` and `draft`, and by any genre
playbook working at sentence level. This is **triage, not dogma** — read
[`voice-guard.md`](./voice-guard.md) before cutting anything that carries
meaning.

## The tightening pass

Run these in order on a paragraph:

1. **Find the buried action.** Underline every *to be* verb (is/are/was) and
   every nominalization (`-tion`, `-ment`, `-ance` nouns made from verbs).
   Each usually hides the real verb. Rebuild as `[who] + [active verb]`:
   "the implementation of X was done by the team" → "the team built X."
2. **Collapse preposition strings.** Three or more chained `of/in/by/to`
   phrases signal bloat; cut or restructure.
3. **Cut clutter.** Delete words that repeat an adjacent word, throat-clearing
   windups ("it should be noted that", "in order to"), and qualifiers that add
   nothing (*very, rather, quite, actually, basically*).
4. **Strip anxious hedging.** Remove metadiscourse that signals nerves rather
   than meaning ("I would argue that", "it seems somewhat"). Keep hedges that
   encode real calibrated uncertainty — that distinction is `voice-guard`'s job.
5. **Prefer the concrete.** Swap abstract nouns for specific people, things,
   and actions the reader can picture.

## Ordering and flow

- **Given before new.** Start a sentence with what the reader already knows;
  end with the new payload. Consecutive sentences should chain old→new.
- **One who-did-what per sentence.** If a sentence carries two unrelated
  actions, split it.
- **Consistent topic strings.** Keep the same grammatical subject across a
  run of sentences about the same thing; silent subject-switching disorients.

## Report the cut

When tightening, report a **before/after word ratio** (e.g. "62 → 41 words,
−34%"). It makes the edit visible and keeps the pass honest — a pass that
cuts 3% probably skipped step 1.

## Quick diagnostic

- Read the paragraph aloud. Where you stumble, the sentence needs work.
- Cover everything but the first sentence — does the reader know what this is
  about? If not, the topic is buried.
