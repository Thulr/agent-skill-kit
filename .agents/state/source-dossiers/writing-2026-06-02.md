## Paraphrased Concepts (operational, by cluster)

### Prose & copyediting craft (‚Üí writing-audit core; writing-design draft)
- **Cut clutter** ‚Äî delete words that repeat an adjacent word, throat-clearing
  windups, redundant qualifiers, euphemism. (Zinsser) ‚Äî H
- **Paramedic pass** ‚Äî find buried actions under *to-be* verbs and preposition
  strings; rebuild as `[agent] + [active verb]`; un-bury `-tion/-ment` nouns;
  cut the slow windup; quantify the cut as a before/after word ratio. (Lanham,
  "Paramedic Method"/"Lard Factor"/"Official Style") ‚Äî H
- **Beat the curse of knowledge** ‚Äî you can't un-know what you know, so you
  under-explain; define terms, add concrete examples, reread cold. (Pinker;
  Heath) ‚Äî H
- **Classic style + coherence** ‚Äî direct the reader's gaze at something
  concrete; order information given-before-new (known first, new last); keep
  topic strings consistent; make connectives explicit. (Pinker) ‚Äî H
- **Copyedit discipline** ‚Äî ban wan intensifiers/throat-clearers on a dedicated
  pass; default to the serial comma; separate real clarity rules from taste
  "peeves"; edit *to the writer's intent*, not to flatten voice. (Dreyer) ‚Äî H
- **Permission to draft badly** ‚Äî write a rough first pass without judging;
  shrink the task to the next small unit; separate generating from editing.
  (Lamott) ‚Äî H

### Structure & argument (‚Üí writing-design core; writing-audit diagnose)
- **Answer-first (top-down)** ‚Äî state the governing conclusion before support;
  invert discovery order (bottom-up) when communicating (top-down). (Minto) ‚Äî H
- **SCQA intro** ‚Äî Situation (agreed context) ‚Üí Complication (the change/
  trigger) ‚Üí Question (what it raises) ‚Üí Answer (your apex thesis). (Minto) ‚Äî H
- **Vertical Q&A + MECE grouping** ‚Äî each point answers the question the point
  above provokes; horizontal groups are Mutually Exclusive, Collectively
  Exhaustive (3‚Äì5 items); pick deductive *or* inductive within a group; the
  group label must assert the insight, not count items. (Minto) ‚Äî H
- **Task orientation** ‚Äî organize by what the user is trying to accomplish, not
  by features; do audience+task analysis first; decompose into ordered
  one-action steps; write minimally; separate procedure from reference.
  (Barker) ‚Äî H
- **Fence vs docs-***: Minto = intra-document rhetoric (order the argument
  inside one artifact); Di√°taxis = corpus architecture (split a doc *set* into
  modes). They compose; answer-first is *wrong* for a learning tutorial. ‚Äî H

### Persuasion & presentation (‚Üí writing-design persuade; writing-audit stickiness)
- **SUCCESs check** ‚Äî Simple (one core idea), Unexpected (open a curiosity
  gap), Concrete (specific people/things, not abstractions), Credible
  (self-testable / one telling number), Emotional (one identifiable person),
  Stories (a mental rehearsal). (Heath) ‚Äî H
- **What-is / what-could-be oscillation** ‚Äî build the spine by contrasting
  present reality with a better future, ending on the new state; plant one
  memorable peak (STAR moment). Presentation-native; adapt for prose. (Duarte) ‚Äî H/M
- **Audience-as-hero, you-as-guide** ‚Äî write from the reader's aspiration;
  run the message through a character‚Üíproblem‚Üíguide‚Üíplan‚Üícall‚Üísuccess/avoid-
  failure arc; compress to a one-liner; pass the "grunt test" (a stranger gets
  it in seconds). (Miller, SB7) ‚Äî H
- **ABT scaffold** ‚Äî `[context] AND [context], BUT [tension], THEREFORE
  [action]`; the smallest engine of the present‚Üîfuture turn; diagnose
  "too many ANDs" (flat list) vs "too many BUTs" (muddle). (Olson) ‚Äî H
- **Pixar Spine** ‚Äî 5‚Äì7 line skeleton: status quo ‚Üí routine ‚Üí disruption ‚Üí
  *because of that‚Ä¶* (causal chain) ‚Üí until finally ‚Üí new normal; strip the
  fairy-tale phrasing. (Adams) ‚Äî H

### Narrative craft ‚Äî work-writing lens (‚Üí light; diagnose + narrate)
- **Open on the gap / turning point** ‚Äî set expectation, deliver the
  uncooperative result; build a section around the moment a value flips, not a
  flat chronology. (McKee) ‚Äî H
- **Does this section turn?** ‚Äî diagnostic: every unit should move a named
  value (confidence‚Üídoubt, ignorance‚Üíinsight); a unit where nothing changes
  state is a cut/merge candidate. Five Commandments (inciting incident ‚Üí
  progressive complication ‚Üí crisis ‚Üí climax ‚Üí resolution) as an audit
  checklist. (Coyne, Story Grid) ‚Äî H
- **Desire vs need + moral argument** ‚Äî give a piece a throughline: the visible
  goal vs the deeper misconception it fixes; dramatize competing approaches
  rather than asserting the theme. (Truby) ‚Äî M/H
- **Causal necessity + reversal/recognition** ‚Äî each point follows the prior as
  a necessary/probable consequence (cure for "and then, and then"); stage a
  reversal (peripeteia) and a recognition (anagnorisis); cut anything whose
  removal leaves the whole intact (unity). (Aristotle ‚Äî public domain) ‚Äî H
- **Cognitive hooks** ‚Äî open with a concrete human problem + a protagonist with
  a goal in a worsening situation; make stakes personal; keep forward motion.
  (Cron ‚Äî use as framing; neuroscience contested) ‚Äî M
- **Transformation arc** ‚Äî ordinary world ‚Üí disruption/call ‚Üí trials ‚Üí return
  changed; cast the customer/team (not the company) as hero. (Campbell ‚Äî note
  Eurocentric/androcentric critique; Snyder pacing ‚Äî paraphrase only) ‚Äî M

## Reusable Behaviors (imperative capabilities ‚Üí candidate skill / intent)

writing-design:
- **Outline an argument answer-first** from a messy brain-dump: extract the one
  governing thesis, build the SCQA intro, group support MECE. (Minto) ‚Üí design/structure
- **Turn a feature list into a task-oriented structure**: map user goals,
  decompose to ordered steps, split procedure from reference. (Barker) ‚Üí design/structure
- **Generate a sustainable draft plan**: chunk the work, produce a deliberately
  rough first pass, defer editing. (Lamott) ‚Üí design/draft
- **Manufacture narrative momentum on demand**: drop in an ABT spine or Pixar
  skeleton; open a present‚Üîfuture gap; plant one STAR moment ‚Äî *only where the
  tension is real*. (Olson/Adams/Duarte/Heath) ‚Üí design/persuade
- **Cast audience-as-hero and pressure-test clarity** with a one-liner + grunt
  test. (Miller) ‚Üí design/persuade

writing-audit:
- **Run a clarity/clutter pass** (Paramedic Method): flag to-be verbs, buried
  agents, nominalizations, preposition strings, wan intensifiers; rewrite
  active+concrete; report a before/after tightening ratio. (Lanham/Zinsser/Dreyer) ‚Üí audit/revise
- **Run a copyedit pass**: mechanics, serial comma, the writer's pet-word tics;
  flag real rules vs taste; preserve voice. (Dreyer) ‚Üí audit/copyedit
- **Diagnose flat structure**: apply "does this section turn?" + Five
  Commandments + causal-necessity test; locate the missing answer-first thesis;
  flag manufactured/absent tension. (Coyne/Aristotle/Minto/McKee) ‚Üí audit/diagnose
- **Score stickiness/persuasion gaps**: SUCCESs checklist + curse-of-knowledge
  scan against a target reader. (Heath/Pinker) ‚Üí audit/diagnose

## Critical / Dissenting Takes (required)

- **Concision can flatten voice.** Lanham/Zinsser's aggressive cutting, applied
  mechanically, strips legitimate hedging, register, and cadence ‚Äî Dreyer and
  Pinker defend voice and "rule-breaking that aids clarity." *The skill must
  treat the paramedic pass as triage, not dogma, and keep a "preserve intended
  voice/nuance" guard.* (sheseeksnonfiction; arrantpedantry; Goodreads)
- **Frameworks breed sameness.** Save the Cat! (Slate/Suderman), StoryBrand
  (uncommoncrowd), Minto (funblocks/youexec), and the monomyth (LARB/bigthink)
  are each blamed for homogenized, formulaic output and, for Minto, a
  corporate-deductive bias that presupposes you already have the answer. *Mine
  the principle; never ship the named sequence as a fill-in-the-blanks mandate.*
- **Fiction structure on non-fiction can manufacture conflict / pad / manipulate.**
  The single recurring meta-critique across all narrative sources. *Every
  narrative move in the skill carries the caveat: only where the tension is
  genuine; do not invent villains or stakes.*
- **Pop-neuroscience as authority.** Cron's brain-science (and Heath's reliance
  on the lone tappers/listeners study) is thin/post-hoc. *Use as motivational
  framing; do not present as evidence.*
- **Presentation-native ‚âÝ page-native.** Duarte's props/peaks/oscillation lose
  force in prose. *Port the persuasive shape; flag the stagecraft as
  out-of-scope for written artifacts.*
- **Monomyth carries bias.** Eurocentric/androcentric source-selection critique;
  the safe transfer is the generic departure‚Üíinitiation‚Üíreturn arc + the
  customer-as-hero adaptation, attributed carefully.
