# Narrative Playbook

## Scope

- **In:** long-form nonfiction that carries the reader through a sequence — an
  essay, a case study, a vision or change narrative, a technical narrative, a
  post-mortem told as a story.
- **Out:** a procedure (→ `technical-doc.md`), a decision memo (→
  `argument-memo.md`), a short pitch or talk (→ `talk-pitch.md`).
- **Intents this surface answers:** `persuade`, `draft` (design); `revise`,
  `diagnose` (audit).

## Grounding

- *Story* (Robert McKee, 1997) — the gap between expectation and result; the
  turning point where a value flips, as the spine of a section.
- *The Story Grid* (Shawn Coyne, 2015) — the editor's test "does this section
  turn?"; the five-part section check (disturbance, complication, crisis, turn,
  resolution).
- *The Anatomy of Story* (John Truby, 2007) — the throughline: the visible goal
  versus the deeper need it actually addresses.
- *Poetics* (Aristotle) — causal necessity over episodic listing; reversal and
  recognition.
- *Wired for Story* (Lisa Cron, 2012) — open on a concrete problem and a goal in
  a worsening situation (cognitive framing, not evidence).
- *The Hero with a Thousand Faces* (Joseph Campbell, 1949) — the departure→
  initiation→return arc for a transformation narrative (the generic arc only;
  the staged template carries known bias).
- The Story Spine (Kenn Adams, 1991) — a five-line setup→disruption→*because of
  that*→resolution skeleton.

> This is the highest-risk genre for misapplied technique. Every heuristic here
> is gated by `narrative-honesty-guard.md`: structure only where the tension is
> genuine.

## Good signals

- Opens on a concrete situation with a clear stake, not an abstract preamble.
- Each section moves a named value start→end; the reader can say what changed.
- Events connect by cause and effect ("because of that"), not mere sequence.
- A real reversal or recognition drives the central insight.
- The piece earns a self-revelation — the reframe the journey was for.

## Common failures

- **Manufactured conflict** — invented stakes or a villain where the situation
  is mundane; the reader senses it and disengages.
- **Episodic recap** — "we did this, then this, then this"; nothing turns, so it
  reads as a log, not a narrative.
- **No turning point** — the section ends in the same state it began; it can be
  cut or merged.
- **Template sameness** — a named framework stamped on beat by beat; the arc
  feels formulaic because it was imposed, not found.
- **Padding for arc** — filler added to hit a shape the material doesn't have.

## Heuristics

- **(draft, persuade) Open concrete** — start on a specific person, problem, and
  goal in a worsening situation, not an abstract setup.
- **(persuade) Turning point as spine** — build each section around the moment a
  value flips (safe→risky, working→broken, assumed→disproven).
- **(persuade) Desire vs. need throughline** — separate the visible goal from
  the deeper misconception it fixes; address both.
- **(draft, persuade) Five-line skeleton** — sketch status quo → disruption →
  *because of that…* → resolution before drafting; strip the fairy-tale phrasing.
- **(diagnose) Does this section turn?** — walk each unit; if no named value
  shifts, mark it cut-or-merge.
- **(diagnose) Causal-necessity test** — each point should follow the prior as a
  consequence; flag "and then, and then."
- **(revise) Cut episodic filler** — remove beats that don't advance the value
  or the cause-and-effect chain.
- **(diagnose, persuade) Honesty check** — for every dramatic beat, confirm the
  tension is already in the material; if not, cut it (see guard).

## Quick diagnostic

| Question | If no | Action |
| --- | --- | --- |
| Does it open concrete, with a stake? | Reader doesn't engage | Replace the preamble with a specific situation |
| Does each section turn? | Reads as a log | Cut or merge static sections |
| Is the causation real? | Episodic "and then" | Make cause-and-effect explicit, or restructure |
| Is the tension genuine? | Manipulation/padding | Remove the manufactured beat |
| Is there a payoff reframe? | Ends flat | Land the self-revelation the journey earned |
