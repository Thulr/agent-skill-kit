## 1. Research question

One sentence. State what is being researched, and explicitly what is
**not** in scope.

- **In scope:** …
- **Out of scope:** …
- **Audience / use:** who this report is for and what decision (if

## 2. Search strategy

What was searched and how. Anyone reading this should be able to
judge the report's coverage and reproduce the search if needed.

- **Source types consulted:** peer-reviewed / standards / industry
  reports / primary sources / expert blogs / vendor docs / …
- **Search terms:** the actual queries used.
- **Databases / engines:** web search engines, specific archives,
  citation graphs used.
- **Snowballing:** anchor sources used for forward/backward citation
  chasing (deep-dive only).
- **Exclusions:** what was deliberately filtered out and why.
- **Stop criterion:** what made the search "done" (new sources stopped
  adding new claims / time budget / source-count target / decision
  deadline).

## 3. Background

Foundational concepts and terminology a reader needs before the rest
of the report makes sense. Define terms on first use. Cite definitions.

## 4. Current state

What is known, established, or widely accepted. Each load-bearing
claim is followed by a citation and a confidence tag.

- **Claim:** … *[source, year]* — confidence: H/M/L
- **Claim:** … *[source, year]* — confidence: H/M/L

## 5. Key debates and open questions

Where the field disagrees, where evidence is thin, where positions
are contested. Distinguish:
- **Empirical disagreement:** sources disagree on what the data shows.
- **Interpretive disagreement:** sources agree on data but disagree
  on what it implies.
- **Open questions:** what no source addresses yet.

## 6. Implications

Practical takeaways, *if applicable*. Skip this section if the topic
is purely descriptive and no implications follow from the evidence.

Mark each implication explicitly:
- **Sourced implication:** directly stated by a cited source.
- **Inferred implication:** synthesized from multiple sources — name
  which sources support the inference.

## 7. Sources

Annotated bibliography. For each:
- Citation (author, year, title, venue/publisher, link if available).
- **Why included:** one sentence on what this source contributes.
- **Source quality:** primary / secondary / tertiary; peer-reviewed /
  institutional / editorial / vendor-aligned.
- **Recency:** date relative to topic — fresh / recent / dated /
  seminal.

## 8. Limitations

Explicit, not buried. A trustworthy report names its own gaps.

- **Coverage gaps:** what topics or sub-questions were not covered
  and why (out of scope, no sources found, time-limited).
- **Source-type gaps:** what kinds of evidence are missing (e.g.,
  no peer-reviewed sources; no industry data; no recent updates).
- **Confidence floor:** the lowest-confidence claims in this report
  and what would raise their confidence.
- **What would change this report:** the specific evidence that
  would meaningfully revise the conclusions if it surfaced.
